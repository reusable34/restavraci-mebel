# Настройка Sanity Studio для добавления услуг

## Шаг 1: Настройка переменных окружения

Создайте файл `.env.local` в корне проекта со следующим содержимым:

```env
SANITY_PROJECT_ID=your_project_id_here
SANITY_DATASET=production
SANITY_TOKEN=your_token_here
NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id_here
NEXT_PUBLIC_SANITY_DATASET=production
```

## Шаг 2: Получение токена Sanity

1. Перейдите в [Sanity Management Console](https://sanity.io/manage)
2. Выберите ваш проект
3. Перейдите в **API > Tokens**
4. Создайте новый токен с правами **Editor**
5. Скопируйте токен и добавьте в `.env.local` как `SANITY_TOKEN`

## Шаг 3: Запуск Sanity Studio

```bash
npm run studio
```

Studio будет доступен по адресу: http://localhost:3333

## Шаг 4: Добавление услуг

### Вариант 1: Автоматическое добавление (рекомендуется)

1. Убедитесь, что у вас есть токен в `.env.local`
2. Запустите скрипт:

```bash
node scripts/add-services-to-sanity.js
```

### Вариант 2: Ручное добавление

1. Откройте Studio по адресу http://localhost:3333
2. Перейдите в раздел **"Услуга"**
3. Нажмите **"+"** для создания новой услуги
4. Заполните поля:
   - **Название**: название услуги
   - **Описание**: описание услуги
   - **Цена от**: цена в формате "X XXX ₽"
   - **Слаг**: автоматически генерируется из названия

# ProvintageVRN — описание проекта для резюме

## Полный вариант (для детального описания в резюме)

**Проект:** ProvintageVRN — лендинг реставрации мебели  
**Позиция:** Фронтенд-разработчик (Next.js)  
**Период:** 2024 — 2025  
**Статус:** Завершён, работает в продакшене

Разработал лендинг для мастерской реставрации мебели на Next.js с подключением Sanity CMS для управления контентом. Настроил схемы данных для услуг и портфолио, реализовал загрузку контента через GROQ-запросы. Сделал адаптивную вёрстку с использованием Tailwind CSS, добавил интерактивный слайдер "до/после" для демонстрации работ.

Настроил форму обратной связи с отправкой заявок в Telegram-бот и fallback на email. Добавил базовые метатеги для SEO (Open Graph, title, description). Настроил домен provintagevrn.ru, подключил HTTPS через Let's Encrypt, настроил редиректы с www-версии. Настроил Nginx как reverse proxy для Next.js приложения.

Клиент управляет контентом через Sanity Studio — может добавлять услуги, загружать фото работ и менять их порядок. Для удобства настроил простые роли доступа.

**Технологии:** Next.js, React, TypeScript, Sanity CMS (GROQ), Tailwind CSS, Nginx, Linux, Git.

## Что реально сделано (для справки)

### Фронтенд
- Next.js 15 с TypeScript
- Адаптивная вёрстка (Tailwind CSS)
- Интерактивный слайдер "до/после" для портфолио
- Форма обратной связи с валидацией
- SEO: базовые метатеги (title, description, Open Graph)

### Бэкенд/API
- API-роут для обработки формы (`/api/contact`)
- Интеграция с Telegram Bot API
- Fallback на mailto, если Telegram недоступен
- Защита от спама (honeypot-поле)

### CMS
- Sanity Studio для управления контентом
- Схемы: услуги (title, description, price), портфолио (фото с порядком сортировки)
- GROQ-запросы для получения данных

### Деплой
- Настроен домен provintagevrn.ru
- Nginx как reverse proxy
- HTTPS через Let's Encrypt
- Редиректы с www-версии
